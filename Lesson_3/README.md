# Занятие №3. Циклы

На этом занятии мы рассмотрим такие вещи как циклы

## `while`

```python
while <условие>:
    <исполняемый код>
```

```python
while <условие>:
    <исполняемый код>
else:
    <исполняемый код>
```
## `for`

```python
for <переменная> in <функция>:
    <исполняемый код>
```

```python
for <переменная> in <функция>:
    <исполняемый код>
else:
    <исполняемый код>
```
